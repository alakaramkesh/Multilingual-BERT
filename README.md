# mBERT UD POS Tagging Project

This project fine-tunes multilingual BERT (`bert-base-multilingual-cased`) for POS tagging on Universal Dependencies treebanks and evaluates cross-lingual transfer between five languages.

Selected languages:

- French Sequoia (`fr`)
- Spanish AnCora (`es`)
- Italian ISDT (`it`)
- English EWT (`en`)
- German GSD (`de`)

## Setup

```bash
pixi install
```

## Run the pipeline

```bash
pixi run dvc repro
```

The main stages are:

```bash
pixi run python src/download_data.py
pixi run python src/warmup_analysis.py
pixi run python src/prepare_datasets.py
pixi run python src/train_all.py
pixi run python src/evaluate_matrix.py
```

## Useful outputs

- Raw UD files: `data/raw/`
- Processed HuggingFace datasets: `data/processed/`
- Fine-tuned models: `models/<language>/best/`
- Corpus statistics: `results/corpus_stats.csv`
- Truncation statistics: `results/truncation_stats.csv`
- Development results: `results/dev_results.csv`
- Transfer matrix: `results/transfer_matrix.csv`
- Markdown transfer matrix: `reports/transfer_matrix.md`

## Notes

The repository does not track raw data, processed datasets, model checkpoints, or generated results directly. These are generated by DVC stages.

The label map uses contiguous ids for real POS labels and reserves `-100` for `<pad>`, so padding tokens, special tokens, and continuation subtokens are ignored by the loss and accuracy computation.
